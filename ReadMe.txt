SOFTWARE REQUIREMENTS:
----------------------

1. Java 6 or Above
2. Any Java EE Application/Web server
3. Ant build tool
4. My SQL 5.5

STEPS TO INSTALL
----------------
1. Install My SQL Server and start it
2. Execute dbscript.sql on MY SQL
3. Edit <PROJECT_HOME>/src/dbinfo.properties file to provide data base info.
4. Execute PROJECT_HOME>ant from command prompt to build project.
5. Deploy the smelite.war from the PROJECT_HOME to any of the Java EE server.

USING APPLICATION
------------------

1. Click on "Vendors" menu option to view all the vendors.
2. Click on "Create Vendor" menu option to create vendor.
3. Click on "Projection Report" to projection report.
